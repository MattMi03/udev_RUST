# udev_RUST

`udev_RUST` 是一个用 Rust 编写的类 udev 设备管理系统，目标是兼容传统的 `.rules` 规则文件，独立于 systemd 运行。项目专注于对 Linux 设备事件（如 USB 插拔）进行实时监听、规则匹配与自动化响应，适用于嵌入式系统、自定义发行版或需要精细化设备控制的场景。

## ✨ 项目目标

- 使用 Rust 实现安全、高性能的设备管理守护进程
- 支持解析 udev 规则（如 `SUBSYSTEM`、`ATTR`、`RUN`、`MODE` 等）
- 实现 `udevd` 守护进程，响应内核发送的 Netlink 设备事件
- 提供 `udevadm` 命令行工具，用于规则测试与事件管理
- 完全独立于 systemd，适合定制化 Linux 系统

## 🚧 开发状态

项目仍在开发中，当前已完成部分模块：

- 内核设备事件监听（基于 Netlink）
- `.rules` 规则文件解析器
- 规则匹配机制
- 基本规则动作执行（如创建设备节点、设置权限、运行命令）
- 守护进程初步逻辑
- 命令行工具初步框架

测试模块正在规划中。
