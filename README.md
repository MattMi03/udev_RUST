## ✨ 项目简介

`rust_udev` 是一个基于 Rust 的用户空间工具，模仿 Linux 系统中的 `udev`，用于：

- 监听内核设备事件（uevent）
- 获取设备属性（类似 `udevadm info`）
- 按规则匹配事件，自动执行命令

项目目标是在 **不依赖 `systemd` 和原生 `udev`** 的环境中实现类似功能，适用于容器、嵌入式设备、极简系统等。

---

## 🔧 当前功能

- ✅ 监听内核设备事件（基于 Netlink）
- ✅ 查询设备属性（模拟 `udevadm info`）
- ✅ 加载规则文件，支持属性匹配 + 命令执行
- ✅ 支持规则热加载（自动监听文件变化）
